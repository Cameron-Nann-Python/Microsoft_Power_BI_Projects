# Word Population Forecast 2021

## Overview
In this Power BI report, I generated a line graph that forecasts the world population from 2021 to the next 10 years at a 95% confidence interval. The line graph was formatted to match the preexisting visuals using the format painter and forecasting line was implemented by using the analysis tab of the visualizations pane.

## Key Insights
- The world population is projected to be 8.3 billion in 2031.
- The upper bound of the confidence interval is 8.4 billion for 2031.
- The lower bound of the confidence interval is 8.1 billion for 2031.
